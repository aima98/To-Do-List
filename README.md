# 📖 [To_Do_List] <a name="about-project"></a>

**[To_Do_List]** is a project where I built a simple HTML list of To Do tasks. The list is styled according to the specifications listed in this lesson(Microverse program). This simple web page was built using webpack and served by a webpack dev server.
